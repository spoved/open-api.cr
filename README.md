# open-api

A crystal implementation of the OpenAPI spec data structures. 

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     open-api:
       github: spoved/open-api.cr
   ```

2. Run `shards install`
